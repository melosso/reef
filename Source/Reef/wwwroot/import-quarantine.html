Quarantine Help

What is Quarantine?

Quarantine is a holding area for rows that failed to import due to validation errors, constraint violations, or other issues. When an import encounters an error with the "Quarantine" error strategy, the problematic row is automatically moved to quarantine for manual review and intervention.

Row Status

  • Pending Review - New row in quarantine, waiting for review
  • Reviewed - Row has been reviewed and notes added
  • Resolved - Row has been fixed and successfully re-imported

Next Steps

  1. Review the quarantined row and understand the error
  2. Check the row data and error message
  3. Fix the issue in the source system or data
  4. Mark the row as reviewed
  5. Delete the row from quarantine or re-import after fixing
