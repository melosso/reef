Import Executions Help

What is an Execution?

An Execution is a single run of an import profile. Each time an import profile is triggered (manually or via schedule), a new execution record is created to track the progress and results.

Understanding Status

  • Running - Execution is currently in progress
  • Completed - Execution finished successfully
  • Failed - Execution encountered errors and failed
  • Canceled - Execution was manually canceled

Key Metrics

  • Rows Read - Total rows fetched from the source
  • Rows Written - Rows successfully written to destination
  • Rows Failed - Rows that encountered errors
  • Rows Skipped - Rows skipped by delta sync or validation

Viewing Details

Click on any execution row to view detailed information including:

  • Execution timeline
  • Full execution logs
  • Error details with row-level information
  • Performance metrics
