Import Profiles Help

What is an Import Profile?

An Import Profile defines how to fetch data from a source (REST API, S3, FTP, Database) and write it to a destination (Database, File, S3). Profiles can run on a schedule or be triggered manually.

Key Features

  • Multiple Sources: Import from REST APIs, AWS S3, FTP/SFTP, or Databases
  • Delta Sync: Only import changed or new rows since the last run
  • Field Mapping: Transform and map fields from source to destination
  • Validation: Validate data before importing (regex, min/max, enums)
  • Error Handling: Choose how to handle errors: Skip, Fail, Quarantine, or Retry
  • Scheduling: Run imports automatically on a schedule (Cron, Interval, Daily, Weekly)

Getting Started

  1. Click New Profile to create an import
  2. Configure the Data Source (where to import from)
  3. Map fields from source to destination (optional)
  4. Set the Destination (where to import to)
  5. Configure error handling and advanced options
  6. Enable scheduling if you want the import to run automatically
  7. Click Save Profile to create the import
