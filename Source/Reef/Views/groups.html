How to use the Groups Page

The Groups page allows you to organize your profiles into logical collections for easier management and execution.

Things to know:

  • Create Group: Use the "New Group" button to define a new group with a name and optional description.
  • View Profiles: The Profiles column shows how many profiles are currently associated with each group.
  • Edit/Delete: Use the (Edit) and (Delete) actions to manage your existing groups.

Tip: Groups are essential for bulk operations on your profiles in the Jobs section.

How to use Groups

Groups are simply categories that help you organize your profiles. Use groups to keep related profiles together for easier management and bulk actions.

How to use Groups:

  • Create Group: Click "New Group" to add a new category. Enter a name and (optionally) a description.
  • Profiles: The Profiles column shows how many profiles are assigned to each group.
  • Edit/Delete: Use the (Edit) and (Delete) icons to update or remove groups.

Tip: Groups make it easy to select sets of profiles for jobs or bulk operations. You can edit or reorganize groups anytime as your needs change.
