How to use Destinations

The Connections page lets you manage and configure your database and data source connections. These connections are used by profiles and jobs to access data for processing and export.

Things to know:

  • Add Connection: Click the "Add Connection" button to create a new connection by specifying its name, type (such as SQL Server, MySQL, or PostgreSQL), and connection string.
  • Edit/Delete: Use the (Edit) and (Delete) icons to modify or remove existing connections.
  • Status: Each connection displays whether it is Active or Inactive, and shows the last time it was tested.
  • Test Connection: Use the Test Connection button in the add/edit form to verify your connection details before saving.
  • Tags: Optionally, add comma-separated tags to help organize and filter your connections.
  • Security: Connection strings are encrypted automatically for your security.

Tip: Reef supports multiple connection types. Make sure your connection string is correct and has the necessary permissions for data access.
